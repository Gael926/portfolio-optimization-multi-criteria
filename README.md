# Optimisation de Portefeuille

Ce projet vise à optimiser un portefeuille d'actifs en utilisant des techniques d'apprentissage automatique et d'optimisation multicritère.

## Structure du projet

- `notebooks/`: Notebooks Jupyter pour l'exploration, l'entraînement et l'analyse.
- `src/`: Code source pour la préparation des données, les modèles et les métriques.
- `model/`: Modèles entraînés et configurations.
- `reports/`: Rapports générés et figures.
- `mlruns/`: Logs MLflow.

## Installation

```bash
pip install -r requirements.txt
```
